<h2 align="center">Aprendizados nesse projeto 🎯</h2>
<p>Trata-se de um jogo da memória com a temática do seriado FRIENDS. </p>

<p>Onde foi possível usar a funcionalidade aspect ratio para criar elementos flexíveis que mantêm sua proporção quando redimensionados.<br>
...</p>

<p> O objetivo é continuar aprofundando os conhecimentos em HTML5, CSS3 e JavaScript.</p>
